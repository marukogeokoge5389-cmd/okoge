# 京都市体育館 予約・抽選申込ツール

京都市施設予約システム（g-kyoto）の予約・抽選申込を自動化するWebアプリです。

## セットアップ（Docker）

### 1. Docker Desktopをインストール
https://www.docker.com/products/docker-desktop/ からダウンロード・インストール

### 2. 起動
```bash
cd kyoto-gym-reservation
docker compose up --build
```
初回はビルドに5〜10分かかります。

### 3. ブラウザでアクセス
http://localhost:5000

### 4. 停止
ターミナルで `Ctrl+C`

---

## 機能

### 予約（http://localhost:5000）
- お気に入り登録済み施設の一括予約
- 1時間単位の時間帯選択

### 抽選申込（http://localhost:5000/lottery）
- バドミントン / 卓球の競技選択
- 2時間単位の時間帯選択（複数同時チェック可）
- 土日祝日のみ日付選択可
- 固定パラメータ: 利用目的=練習 / 入場者数=20人

## 対応施設

| 競技 | 体育館 |
|------|--------|
| バドミントン | 醍醐地域体育館（1/2面・全面） |
| 卓球 | 下京地域体育館（1/4面） |
